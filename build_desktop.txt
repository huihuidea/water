Desktop packaging notes

Recommended files in this folder:
- requirements_desktop.txt
- watermark_desktop.spec
- build_desktop.command

macOS quick build:
1. chmod +x build_desktop.command
2. ./build_desktop.command

Manual macOS build:
python3 -m pip install --user -r requirements_desktop.txt
python3 -m PyInstaller --noconfirm watermark_desktop.spec

Windows build:
py -m pip install -r requirements_desktop.txt
py -m PyInstaller --noconfirm watermark_desktop.spec

Output:
- macOS app bundle: dist/AIWatermarkTool.app
- Windows executable folder: dist/AIWatermarkTool/

Notes:
- PDF document support needs PyMuPDF.
- `.docx` support uses only Python standard library + XML/zip processing.
- The desktop app now keeps only Word/PDF watermark removal.
